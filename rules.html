부동산 관리 DB 규칙 + 인터페이스 + 사용자권한

📜 핵심 DB 규칙 (1~13)

1주소 기반 데이터베이스
2핵심키는 별도 고유 ID — 주소는 속성일 뿐 (같은 주소라도 층수/호실 다르면 독립된 데이터)
3증가는 가능, 삭제는 불가능
4부동산 특성에 따라 테이블값 형식 특정
5데이터베이스 수정 가능
6동일 주소 + 동일 층수/호실 조합에서만 중복 불가
7DB 직접 조작 시 로그 기록 필수
8수정 시 신규 INSERT + 기존 비활성화 (DELETE 금지)
14KST 한국타임 기준으로 모든 테이블의 timestamps 통일
15DB 간 연결: properties.id = contracts.property_id, contracts.tenant_ids(JSON) ↔ tenants.id 등. 앱 로직에서 자동 조인
~중요한 사항들입니다.

실무상 중개사 개입이 많은데 중개료는 계약에 따라 변동사항으로 별도 테이블로 관리해야 함 (rules.txt에는 기본 규칙만)
계약서 입력 시 양측(임대인·임차인) + 중개인 정보 함께 채워야 하고
계약서 원본이미지는 상시 visible / 옆쪽에 뜨게 만들어야 해요.

👥 사용자 권한+감사로그 규칙 (16~19)

16관리자(거의 전권): 모든 데이터 조작권한 (불특정사용자의 접근금지)
17사무인원: 검색+입력 권한 (수정불가능, 삭제불가능)
18건물유지보수인원: 시설만 입력 + 조회 (계약서수정불가)
19누가 어디에 무슨 짓을 했는지 로그파일로 관리 + 감사로그테이블 (db_logs)

🖥️ 구현할 인터페이스 목록 (구현우선순위)
각 항목 입력/수정/검색 모두 별도 구현 필요, 계약서 입력시는 계약자정보도 같이 채워야 하고
계약서 원본은 항상 옆에 떠 있도록

A. 부동산 관리

입력 — 부동산 등록 (주소·층수·호실·유형(상가등)면적)

B. 계약서 관리

입력 — 계약조건 입력, 계약서 원본업로드
수정 — 조건변경 (이력관리), 기존 비활성화+신규INSERT

C. 계약자 관리

입력 — 성명/연락처/사본증명(주민등록 앞자리만)
수정 — 개인정보변경 (삭제는 없음), 이력관리

D. 관리공과금(관리비)

입력 — 재산세/전기/수도/난방/통신 고지금액/납부기한

E. 월세 납부 관리

입력 — 월세납입일·금액(보증금반환도 포함)

F. 시설유지보수(INCIDENTS)

파손신고,repair필요사항/비용추산

G. 통합 월세 관리 인터페이스

부동산별 계약자별 월세총수입·미납목록·연체일수 통합현황판
월세+공과금+보증금반환을 한눈에 확인할 수 있음

H. 통합 계약 관리 인터페이스

모든계약의만기·갱신·폐지현황 — 다음행위추적표

부동산총수·계약현황·미납월세/예외사건(파손등) 통합현황판
모든 데이터를 요약하고 한눈에 확인하는 직원용 메인화면

/각 항목 입력/수정/검색모두 별도구현 필요, 계약서입력시는 계약자정보도 같이 채워야 하고
계약서 원본은 항상 옆에 떠 있도록

A.부동산 관리

입력 — 부동산 등록 (주소·층수·호실·유형(상가등면적)
수정 — 현황변경, 면적확장 특성변경삭제 없음

검색 — 주소/유형별현황리스트

B. 계약서관리

입력 — 계약조건 입력 (보증금·임대료·기간·전월세형태), 계약서 원본업로드
수정 — 조건변경(이력관리)기존 비활성화+신규INSERT

검색 — 부동산별 계약리스트 / 기간에 갱신표 198.0, 93.0, 2025-04-15T00:00:00Z

C. 계약자관리

입력 — 성명·연락처·사본증명(주민등록앞자리만) 등록
수정 — 개인정보변경삭제이력, 이력관리)

검색 — 계약자이력표 / 계약현황/연체여부

D.공과금 관리

입력 — 재산세/수도/난업·통신고지금액·납부기한 등

수정 — 금액변경/납부확인(이력관리) / 검색미납현황

E. 월세납입 관리

입력 — 월세납입일·금액(보증금 반환포함)
수정 — 납부이력변경 연체처리 이역 (삭제< 없음))

검색 — 월별납현황/총계요약 / 미납자리스트

F.시설유지보수(INCIDENTS)

입력 — 파손신고·필요사항/비용추산

수정 — 처리상태변경 (접수중→완료)/실수리비정산

검색 — 부동산별유지이력 / 파손별비용추계

G.통합월세 관리 인터페이스

부동산별 계약자별 월세총수입/미납목록·연체일수
→ 월세+공과금+보증금반환을 한눈에

H. 통합계약 관리 인터페이스

모든계약의만기·갱신폐지현황 / 다음행위추적표
→ 계약서+계약자+ 부동산연결고리 한눈에

부동산총수/계약현황·미납월세/예외사건파손등 통합현황
→ 모든 데이터를 요약하고 한눈에 확인하는 직원용 메인화면
